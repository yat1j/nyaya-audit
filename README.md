# Nyaya — Frontend

This is the Flutter web app for Nyaya. It gives users a dashboard to run bias audits and view results in real time.

**What it does:**

1. **Upload screen** — User uploads a CSV dataset and clicks Start Audit. The app calls the backend API directly and creates a Firestore job document.

2. **Results screen** — Shows 4 bias metric cards (Caste Bias, Religion Bias, Demographic Parity, Fairness Verdict) that auto-update in real time using Firestore StreamBuilder — no manual refresh needed.

3. **Verdict banner** — Shows CERTIFIED COMPLIANT (green) or BIAS DETECTED (red) based on whether demographic parity passes the 0.80 threshold.

4. **Gemini explanation** — Displays the plain-English bias summary generated by Gemini.

5. **t-SNE visualisation** — Interactive before/after toggle showing the geometric shift of name clusters in embedding space across 4 groups: Brahmin (purple), Dalit (yellow), Hindu (green), Muslim (blue).

6. **Audit Trail** — Table showing every applicant's original score, fair score, and whether their shortlisting status changed.

7. **PDF certificate** — Downloadable compliance certificate with all metrics.
